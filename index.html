Work

Here is one of my projects!
Point of Sale System (POS)

This is a music/movie rental shop application software.

Written in Java Language with integrated Java Swing Graphical User Interface (GUI) and SQL Database. With an intuitive design the rent shop staffs can issue rents, view the in-stock and rented titles. Also being able to register, search, update and delete customers and titles.

The system is encapsulated with packages, access modifiers and using the well known MVC design pattern as the system architecture.
